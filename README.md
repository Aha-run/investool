## 我的选股规则

根据各种指标筛选值得长期持有其股票进行投资的优质公司。（优质公司不代表当前股价在涨，长线存股）

### 1. 财务优质

- 最新 ROE 高于 8%
- ROE 平均值小于 20 时，至少 3 年内逐年递增
- EPS 至少 3 年内逐年递增
- 营业总收入至少 3 年内逐年递增
- 净利润至少 3 年内逐年递增
- 负债率低于 60%

### 2. 估值优质

- 估值较低或中等
- 股价低于合理价格（合理价格 = 历史市盈率中位数 _ (EPS _ (1 + 今年 Q1 营收增长比))）

### 3. 低波动

- 行业分散，均衡配置
- 历史波动率在 1 以内（持仓占比： 0.1:0.1-0.5:0.5-1 = 3:3:4 ）
